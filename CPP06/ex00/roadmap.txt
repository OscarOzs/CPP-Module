Le but est de faire un programme utilisant les parametre du main.
Cette chaine de caractere doit etre convertit de sa forme de base (char *) a son type correspondant
et afficher sa valeur dans ces 4 types suivant :
- int
- char
- float (-inff, +inff, nanf)
- double (-inf, +inf, nan)


ROAD MAP :
- verifier si la string envoye est valide (afficher un msg le cas echeant)
- parsing pour identifier le type reel de la string
- Convertir ce type dans les trois autres types


IsInputValid :
- len de l'input == 0
- input != pseudos literaux
- contient autre chose que 'f' '.' '-' '+' et "0123456789"
- verifier que le '-' et le '+' sont bien suivis d'un chiffre et a l'index 0
- verifier que le 'f' est bien le dernier char et est precedé d'un chiffre
- verifier que le '.' est bien precedé et suivis de chiffre
- input > INTMAX (utilisé atol et limit.h)
- input < INTMIN (utilisé atol et limit.h)

- gerer le cas ou il y a '-' et '.'
